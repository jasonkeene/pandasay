
PANDASAY!
=========

Make panda say things! :D

Dependencies
------------

- `cowsay <http://www.nog.net/~tony/warez/cowsay.shtml>`_

Installation
------------

1. Move panda.cow into your COWPATH.
2. alias pandasay="cowsay -f panda"
3. Enjoy :)
